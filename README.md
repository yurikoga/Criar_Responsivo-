# How to Create Responsive Navigation Bar With HTML & CSS in Hindi

Um projeto simples de **barra de navegação responsiva** usando HTML e CSS, com explicação em Hindi.

## ✨ Sobre

Este projeto demonstra como criar uma **navigation bar moderna e responsiva**, ideal para sites, com conteúdo explicado em Hindi.

## 🛠️ Tecnologias

- HTML
- CSS

## 🎯 Funcionalidades

- Menu de navegação responsivo
- Layout adaptável para diferentes telas
- Design simples e limpo
- Conteúdo em Hindi

## 🚀 Como usar

1. Baixe ou clone o repositório  
2. Abra o arquivo `index.html` no navegador
